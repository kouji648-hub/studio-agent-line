## 概要

20〜40代の子育て世代ママをターゲットにした、LINE公式アカウント用コンバージョン最適化フローのHTMLデモ。

## デモURL

## 3つのパターン

### パターンA: 興味ベース（What型）
- **アプローチ**: 「何を撮りたい？」から入る能動的選択
- **フロー**: カルーセル → リッチメッセージ → CTA
- **想定CVR**: 25-30%

### パターンB: タイミングベース（When型）★推奨
- **アプローチ**: 「お子様の状況は？」で自動セグメント
- **フロー**: 状況別分岐 → パーソナライズ提案 → CTA
- **想定CVR**: 30-40%（最高）
- **特徴**: 妊娠中→ニューボーン予約、お宮参り＋ニューボーンセット、七五三の詳細分岐

### パターンC: ニーズベース（Why型）
- **アプローチ**: 「こんなお悩みありませんか？」で共感から入る
- **フロー**: 悩み選択 → 解決提案 → 口コミ → CTA
- **想定CVR**: 20-30%
- **特徴**: 5つの悩み別フロー（緊張、衣装、赤ちゃん泣く、料金、ロケ）
